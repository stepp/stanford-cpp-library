Detailed Description


A Timer object is useful for measuring the elapsed time of a program in milliseconds over a given interval.

Usage example:

  Timer tim;
  tim.start();
  ... code that takes a while ...
  tim.stop();
  cout << "That took " << tim.elapsed() << "ms." << endl;
