Detailed Description

template<typename T>
class Observable< T >

This abstract superclass allows objects to store lists of observers, which are other objects that would like to be notified when some part of the state of the observable object changes.

The intended usage is that you should extend Observable in the object you want to be watched, then call notifyObservers in its code at appropriate places. Then create some other object that extends Observer and defines an update method, and attach it to the Observable so it will be notified.
