1 /*
2  * File: biginteger.h
3  * ------------------
4  * This file exports a class for arbitrary-size integer arithmetic.
5  * It is meant to help get around the max/min value limit for types
6  * such as int and long.
7  *
8  * In general, a BigInteger supports the standard operators and operations
9  * that you would expect to be able to use on an int or long value.
10  *
11  * Example usage:
12  *
13  * BigInteger bi("1234567890123456789");
14  * for (int i = 0; i < 10; i++) {
15  * bi *= 12345678;
16  * }
17  * cout << "really big number is: " << bi << endl;
18  *
19  * Implementation notes:
20  * This code is heavily based on a BigInteger library taken from:
21  * https://github.com/panks/BigInteger
22  *
23  * The implementation stores the big integer as a string of base-10 digits
24  * along with a sign bit represented as a bool.
25  * In general the operations are not heavily optimized; this class should not
26  * be used in situations where performance is critical.
27  * Note that better Big Integer libraries exist in other places for more
28  * serious work and can be found using your favorite search engine.
29  *
30  * @version 2018/09/25
31  * - added doc comments for new documentation generation
32  * @version 2017/10/28
33  * - initial version
34  */
