Detailed Description

An IntRange2D is an iterable contiguous 2-dimensional range of integers.

It can be processed using a for-each loop to emit the range as a series of Point objects in y-major (default) or x-major order. The points are not actually all stored in this object, so it does not require O(WxH) memory usage.
