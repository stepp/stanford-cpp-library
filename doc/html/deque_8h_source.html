32 /*
33  * Class: Deque<ValueType>
34  * -----------------------
35  * This class models a linear structure called a <b><i>deque</i></b>
36  * in which values can be added and removed at either end.
37  * This discipline allows first-in/first-out (FIFO) and/or
38  * last-in/first-out (LIFO) behavior. That is the defining
39  * feature of deques.
40  */

48  * Initializes a new empty deque.
